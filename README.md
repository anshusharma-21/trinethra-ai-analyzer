# Trinethra Pro (Aura Edition) 🛸
**Advanced AI Supervisor Feedback Analyzer**

Trinethra ek futuristic diagnostic tool hai jo supervisor ke transcripts ko analyze karta hai. Ye project special isliye hai kyunki ye **Llama 3 (via Ollama)** ka use karke raw feedback ko professional performance metrics mein convert kar deta hai.

## 🚀 Key Features
*   **AI Intelligence:** Ollama aur Llama 3 model ka use karke deep feedback analysis.
*   **Performance Index:** Ek bada aur clear score (0-10) jo turant performance level batata hai.
*   **KPI Mapping:** Collaboration aur Impact jaise domains ko automatically identify karta hai.
*   **Cyber-Terminal UI:** Iska interface Glassmorphism aur Neon-glow aesthetics par based hai, jo isse ek premium look deta hai.
*   **Export Data:** Click karte hi puri analysis report PDF format mein download ho jati hai.

## 🛠️ Tech Stack
*   **Frontend:** React.js, Tailwind CSS (Styling ke liye).
*   **Backend:** Node.js, Express.
*   **AI Engine:** Ollama (Local LLM Llama 3).
*   **Libraries:** jsPDF, html2canvas (PDF generation ke liye).

## 🏃 Project Kaise Chalayein (Setup)
1.  **Ollama:** Apne system par Ollama install karke `ollama run llama3` command chalayein.
2.  **Backend Setup:**
    - `/backend` folder mein jayein.
    - `npm install` karein aur `node server.js` se server start karein.
3.  **Frontend Setup:**
    - `/frontend` folder mein jayein.
    - `npm install` karein aur `npm start` se dashboard open karein.
